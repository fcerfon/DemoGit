### INSTRUCTION

## Creer un repository sur github appele demo git ou command git

# __Init__ :

```
git init
```
> Initialise un dossier git

# __commit__ :
```
git commit
```
> ~~Fait des chocapics~~
> Permet d'ajouter les modifications a l'historique :shipit:
